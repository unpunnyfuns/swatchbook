# Tokens

The full, unfiltered token graph — every primitive and semantic token
currently loaded by the addon. Variation is carried by the active
`mode × brand` tuple (switch from the toolbar); there is no separate
component-token layer.

## Semantic color

`color.sys.*` tokens alias into `color.ref.*`. Hover any alias in
`TokenDetail` below to see the chain resolve.

`color.sys.accent.bg` → `color.ref.blue.700` (`Default` brand) or
`color.ref.violet.700` (`Brand A`).
